# japan-covid-map

## This is the application for daily COVID-19 cases in Japan.
## This app uses 
- #### React.js (Javascript)
- #### d3.js (Javascript)
- #### FastAPI (Python) as backend

# Backend
## 1. Install Anaconda
## 2. Create new environment
```
cd backend
conda env create -f env.yml
``` 
## 3. Activate the env
```
conda japan-covid-map
```
## 4. Activate server
```
python main.py
```

# Frontend
## 1. Install node
## 2. Open another terminal and install yarn and all dependencies
```
cd frontend
npm install yarn
yarn
``` 
## 3. Start app
```
yarn start
```
